Meet React App

The objective of The Meet App was to build a serverless, PWA (progressive web application). This was built using the TDD (test-driven development) technique with React. We used Jest and Jest-Cucumber for unit and integration testing as well as Puppeteer for end-to-end testing.

Technologies Used

  • React
  • AWS Serverless
  • Google Calendar API
  • Jest
  • Jest-Cucumber
  • Puppeteer
  • GitHub
